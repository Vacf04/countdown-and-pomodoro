# STOPWATCH AND POMODORO

Aplicativo web de pomodo e cronometro, podendo alterar a funcionalidade pelo menu, com um design simples e bonito.

## FEATURES

- Menu responsivo.
- Pomodoro com start e pause.
- Cronometro com start, pause e reset.
